# 着せ替えアプリの複製・改造ガイド

このプロジェクトのソースコードを元に、新しい着せ替えアプリ（例：洋服、ドレス、コスプレ等）を作成する手順です。

## 1. プロジェクトのコピー
現在のフォルダをコピーして別名で保存します。
```bash
cp -r 20260203kimono_generator fashion_app
cd fashion_app
rm -rf .git  # 新しいリポジトリにするために古いGit情報を削除
git init     # 新しく初期化
```

## 2. 画像アセットの準備
`images/` フォルダの中身を入れ替えます。
- ターゲットとなる服を着たマネキン、またはモデルの画像を用意。
- 背景がシンプルなものがAIにとって認識しやすいです。

## 3. コードの修正点

### app.js (データ設定)
`CONFIG.kimonos` の配列を、新しい画像パスとアイテム名に書き換えます。

### app.js (AIプロンプト)
`handleGenerate` 関数内（240行目付近）の `text` プロンプトを、対象のアイテムに合わせて調整します。
- **ポイント**: 「顔を同一に保つこと」「Image 2 の服を忠実に再現すること」という指示は残してください。

### index.html / style.css
- タイトルや配色（和風から洋風へなど）を必要に応じて変更。

## 4. デプロイ
Render で新しい Web Service を作成する際、環境変数 `ACCESS_CODE` をそのアプリ専用のものに変えることで、各アプリごとにパスワードを分けることができます。
